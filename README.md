# SportClub – Tu mejor versión comienza hoy

Sitio web estático desarrollado con HTML5 y CSS3 para el club deportivo **SportClub**, como parte de la Evaluación Sumativa 1 del curso de Programación Front End – INACAP.

---

## Descripción

Sistema web que permite a usuarios registrarse, iniciar sesión, revisar reservas de clases deportivas y acceder a diferentes paneles según su rol (Usuario, Coach o Administrador).

## Módulos desarrollados

- **Landing Page** – Hero, Beneficios, Planes, Sobre el Club, Footer
- **Login** – Formulario con accesos rápidos a dashboards
- **Registro** – Formulario con perfil deportivo y mensajes de validación
- **Recuperar contraseña** – Formulario con mensaje de confirmación visual
- **Dashboard Usuario** – Reservas, clases disponibles y perfil
- **Dashboard Coach** – Alumnos, clases asignadas y horarios semanales
- **Dashboard Administrador** – Estadísticas, usuarios, reportes y configuración
